About Us

Decide es un proyecto de software libre que representa una plataforma de voto electrónico. Está dividida en diferentes módulos independientes conectados mediante API. Estos módulos son base, auth, census, voting, booth, mixnet, store, postproc, visualizer y pueden recibir una serie de mejoras o incrementos.

    User/Password Sign Up

    Podrá registrarse en la plataforma Decide introduciendo sus datos como su correo electrónico, un usuario y una contraseña

    LDAP Log In

    Podrá iniciar sesión mediante el Protocolo Ligero de Acceso a Directorio (LDAP) previamente su organización haya introducido sus credenciales en Decide

    User/Password Log In

    Podrá iniciar sesión mediante su usuario y contraseña previamente registrada en Decide mediante el método registro de usuario.

    API Rest

    El módulo de autenticación de Decide incorpora una API Rest mediante la cual se pueden consultar todos los usuarios existentes, además de mostrar o borrar un usuario mediante su identificador.
